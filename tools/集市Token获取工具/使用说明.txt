校园集市身份获取工具 - 使用说明
=================================

用途
----

Ahu_Plus 的校园集市使用独立 Bearer JWT，不使用安大统一身份认证。本工具在 Windows 本机识别电脑版微信中已经登录的集市身份，生成 Ahu_Plus 可导入的二维码，并保留剪贴板粘贴方式。

Token 等同于集市登录态。只在自己的电脑和账号上使用，不要发送给他人。

运行条件
--------

- Windows 10 或 Windows 11。
- 电脑版微信，且集市小程序已经登录。
- 首次准备运行环境时需要联网。
- 不需要管理员权限；代理兼容模式只安装到当前用户证书库。

使用步骤
--------

1. 双击“获取集市Token.cmd”。
2. 按窗口提示，在电脑版微信打开“校园集市 / 赞噢校园集市”小程序。
3. 在帖子列表下拉刷新，等待工具识别身份，最长 5 分钟。
4. 成功后工具会打开“集市身份导入二维码.png”。
5. 在 Ahu_Plus 的集市设置中选择扫描电脑二维码并确认导入。

Token 还会写入“我的集市Token.txt”并尝试复制到剪贴板，可在集市设置中直接粘贴。

两种工作模式
------------

1. 微信已经运行

工具只读扫描 WeChat / Weixin / WeChatAppEx 进程的可读内存区域，识别包含有效学校信息和过期时间的 JWT。此模式不修改系统代理，也不安装证书。部分安全软件可能阻止进程内存读取。

2. 微信尚未运行

工具启动只允许 api.zxs-bbs.cn 的本地 mitmproxy，保存原系统代理后临时改为 127.0.0.1:8080，并把本次运行生成的 CA 安装到 CurrentUser\Root。结束时会还原代理、按指纹删除证书，并删除临时 CA 私钥目录。

运行环境
--------

工具优先复用 Python 3.10 及以上和已安装的 mitmproxy、qrcode。系统有 Python 但缺组件时，会尝试用 --user 安装。没有可用 Python 时，才会在本目录创建 python/ 便携环境。

运行时下载源和版本由 run.ps1 定义。删除工具目录中的 python/ 即可移除便携环境；使用系统 Python 时，--user 安装的包不在本目录内。

失败排查
--------

- 未识别身份：确认使用电脑版微信，集市小程序已登录，并在帖子列表刷新。
- 安全软件拦截：允许该工具只读访问微信进程，或关闭微信后重新运行以使用代理兼容模式。
- 端口占用：代理模式要求本机 8080 端口空闲。
- TLS 失败：确认当前用户证书安装没有被安全策略拦截。
- 网络异常：正常结束工具会恢复原代理。若进程被强制终止，到 Windows“设置 -> 网络和 Internet -> 代理”恢复运行前的设置。
- 身份格式无法识别：集市后端可能已改变 JWT 结构，需要更新 catch_token.py 和 App 的 MarketApi 解析器。

隐私与清理
----------

- 代理模式通过 --allow-hosts 只解密 api.zxs-bbs.cn。
- 二维码由本机 qrcode 库生成，不调用在线二维码服务。
- 二维码包含一次性 nonce，但仍携带可导入的真实 Token；关闭工具窗口时会删除二维码文件。
- “我的集市Token.txt”不会自动删除，用完后应手动删除。
- .gitignore 已排除 Token、二维码、便携运行时和状态标记。
- 不要把“我的集市Token.txt”、二维码、.captured 或证书标记提交到仓库。

文件说明
--------

- 获取集市Token.cmd：普通用户入口。
- run.ps1：运行环境、内存扫描/代理选择、清理和提示。
- wechat_memory_token.py：已运行微信的只读内存扫描。
- catch_token.py：目标域请求头捕获、Token 文件和二维码生成。
- _mitmrun.py：mitmproxy 启动垫片。
- 使用说明.txt：本文件。
- python/：需要时生成的便携 Python。
- 我的集市Token.txt：成功后生成的敏感文件。
- 集市身份导入二维码.png：成功后生成，正常关闭窗口时删除。
