============================================================
        校园集市 Token 一键获取工具 · 使用说明
============================================================

这是什么？
  Ahu_Plus 的「校园集市」功能需要一段你的登录身份（Token）。
  集市登录被锁在微信小程序里，App 没法自己拿到，所以做了这个
  小工具：在电脑上点一下，自动把 Token 抓出来、复制到剪贴板，
  你回 App 粘贴即可。全程不用懂技术。


------------------------------------------------------------
一、准备
------------------------------------------------------------
  1) 一台 Windows 电脑（Win10 / Win11）。
  2) 电脑版微信，并且能登录你自己的微信。
  3) 网络正常（首次运行会自动下载一个小的运行环境）。


------------------------------------------------------------
二、怎么用（一共 3 步）
------------------------------------------------------------
  第 1 步：双击运行  「获取集市Token.cmd」
           会弹出「是否允许此应用更改你的设备」——点【是】。
           （需要管理员权限来安装抓包证书，仅此一次。）

  第 2 步：按窗口里的提示操作——
           · 打开【电脑版微信】
           · 在微信里搜索并打开【校园集市 / 赞噢校园集市】小程序
           · 在小程序里随便点一下，比如下拉刷新一下帖子列表

  第 3 步：看到「抓到啦！」就成功了。
           Token 已经自动复制到剪贴板，也存到了同目录的
           「我的集市Token.txt」里。
           回到 App →【集市设置】→ 粘贴 → 保存，完成！

  全部抓完后工具会自动还原你的网络设置并关闭，不用管。


------------------------------------------------------------
三、第一次运行可能慢一点（正常现象）
------------------------------------------------------------
  工具会优先使用你电脑上已有的 Python 和抓包组件——如果都齐了，
  几乎是秒开，不用下载任何东西。

  如果你电脑没装 Python，它才会在自己的文件夹里准备一个便携版
  运行环境（下载 Python + 抓包组件），大概几十 MB，按网速可能要
  等 1~3 分钟。这些东西只放在本文件夹，不会动你的系统，删掉文件夹
  即可彻底清除。以后再用就很快了。


------------------------------------------------------------
四、抓不到 / 出问题怎么办
------------------------------------------------------------
  ▸ 提示没抓到？
    - 确认是在【电脑版微信】里打开的集市小程序（手机上的不行）。
    - 打开小程序后要「点一下」让它发生网络请求，比如下拉刷新。
    - 确认你在小程序里已经登录过集市。
    - 重新双击运行本工具再试一次即可。

  ▸ 杀毒/安全软件弹窗拦截？
    本工具要临时设置系统代理、安装一张抓包证书，部分安全软件
    （如某些国产全家桶）会报警。请选择「允许/信任」后重试。
    工具完全本地运行，不会上传你的任何数据。

  ▸ 我在用加速器 / 科学上网（clash、v2ray 等）？
    没关系，工具已做兼容：下载组件时会自动绕开你的代理，
    抓包结束后会把你的网络设置原样还原。若仍异常，可先临时
    退出加速器再运行。

  ▸ 担心断网？
    工具用了「无论成功失败都还原网络设置」的保护。万一中途强行
    关窗口导致上不了网，再完整跑一次本工具让它正常结束，或到
    Windows【设置 → 网络和 Internet → 代理】把「使用代理服务器」
    关掉即可恢复。


------------------------------------------------------------
五、隐私与安全
------------------------------------------------------------
  · 工具只在你本机运行，只抓「集市」一个域名的身份字段，
    不碰也不上传你的其它任何数据。
  · Token 等同于你的集市登录态，请勿发给陌生人。
  · 「我的集市Token.txt」用完可以删除。
  · 想彻底清理：直接删除整个工具文件夹即可。


------------------------------------------------------------
六、文件夹里都是啥
------------------------------------------------------------
  获取集市Token.cmd ... 双击它就行（入口）
  run.ps1 ............. 主程序逻辑（自动调用，别手动点）
  catch_token.py ...... 抓包监听脚本（自动调用）
  _mitmrun.py ......... 抓包启动垫片（自动调用）
  使用说明.txt ........ 本文件
  python/ ............. 仅当你电脑没装 Python 时才自动生成的便携运行环境
  我的集市Token.txt ... 抓取成功后生成，里面是你的 Token
